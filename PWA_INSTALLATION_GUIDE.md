# 📱 Установка CareBot Admin PWA на телефон

## 🔒 Шаг 1: Установка SSL сертификата

Сертификат находится в `C:\Users\staro\Downloads\carebot-cert.crt`

### Android

1. **Скопируй `carebot-cert.crt` на телефон** (через USB, email, или облако)

2. **Установи сертификат:**
   - Открой **Настройки** → **Безопасность** → **Шифрование и учетные данные**
   - Нажми **Установить сертификат из хранилища**
   - Выбери **Сертификат CA** (или "Wi-Fi сертификат")
   - Найди файл `carebot-cert.crt` и выбери его
   - Введи имя: `CareBot Admin`
   - ⚠️ **Важно:** Выбери "Использовать для: VPN и приложения" (НЕ только Wi-Fi)

3. **Проверь установку:**
   - Настройки → Безопасность → Доверенные учетные данные → Вкладка "Пользовательские"
   - Должен появиться `CareBot Admin`

### iOS/iPad

1. **Скопируй `carebot-cert.crt` на устройство** (AirDrop, email, или облако)

2. **Установи профиль:**
   - Открой файл crt - появится уведомление "Профиль загружен"
   - Открой **Настройки** → **Основные** → **VPN и управление устройством**
   - Нажми на профиль `CareBot Admin`
   - Нажми **Установить** (потребуется код-пароль)

3. **Включи доверие к сертификату:**
   - **Настройки** → **Основные** → **Об этом устройстве** → **Настройки сертификата**
   - Найди `CareBot Admin` и включи переключатель

---

## 🚀 Шаг 2: Установка PWA

### Подключись к локальной сети

Убедись что телефон в той же Wi-Fi сети что и сервер (192.168.1.125)

### Открой в браузере

**Chrome/Samsung Internet (Android):**
1. Открой: `https://192.168.1.125:5555/admin/dashboard`
2. Войди в админ-панель
3. Нажми на **⋮** (меню) → **Установить приложение** или **Добавить на главный экран**
4. Подтверди установку

**Safari (iOS):**
1. Открой: `https://192.168.1.125:5555/admin/dashboard`
2. Войди в админ-панель  
3. Нажми кнопку **Поделиться** (квадрат со стрелкой вверх)
4. Пролистай вниз и выбери **На экран "Домой"**
5. Нажми **Добавить**

## ✅ Проверка PWA

После установки:

1. **Иконка на главном экране** - должна появиться иконка "CareBot"
2. **Запуск без браузера** - приложение открывается в отдельном окне (без адресной строки)
3. **Offline работа** - попробуй включить режим полета и открыть приложение

### Тестирование offline режима

1. Открой приложение онлайн
2. Перейди по нескольким страницам (dashboard, create mission)
3. Включи **режим полета**
4. Закрой и открой приложение снова
5. ✅ Страницы должны загружаться из кеша

## 🔍 Устранение проблем

### "Ваше подключение не защищено" / "NET::ERR_CERT_AUTHORITY_INVALID"

- ✅ Сертификат установлен?
- ✅ iOS: Доверие включено в настройках сертификата?
- ✅ Android: Выбран правильный тип использования сертификата?

### PWA не устанавливается

- ✅ HTTPS работает? (должен быть замочек в адресной строке)
- ✅ Service Worker зарегистрирован? (открой DevTools → Application → Service Workers)
- ✅ Manifest доступен? (откройDevTools → Application → Manifest)

### Страницы не кешируются offline

- Откройте Chrome DevTools → Application → Service Workers
- Проверьте что статус "activated and running"
- Проверьте Storage → Cache Storage - должны быть кеши `carebot-admin-v1`, `carebot-runtime-v1`
